Pterodactyl, installer.

One command. Questions asked upfront. Panel, Wings, or both — running in minutes.

Run on your VPS as root

$ bash <(curl -fsSL https://pterodactyl-installer.xento.xyz/install.sh)

# root required  ·  Ubuntu 20/22/24  ·  Debian 11/12  ·  Rocky/Alma 8/9

Why use this installer

Everything handled for you

⚡
One Command
Paste one line and answer a few questions. The rest runs unattended.
📋
All Inputs Upfront
Every question is asked before installation begins — no mid-install pauses.
👤
Single Admin User
Your credentials are the first and only admin account. Simple and clean.
🔒
Auto SSL
Let's Encrypt via Certbot, configured and enabled automatically.
🐳
Panel + Wings Together
Install both on one machine, or on separate servers — your choice.
🗄️
phpMyAdmin Included
Optional phpMyAdmin installer built right in — manage your DB with a GUI.

What you can do

Menu options

[1]
Install Panel
Full panel install — PHP 8.3, MariaDB, Redis, NGINX, SSL, queue worker.
[2]
Install Wings
Docker + Wings daemon. Optional MariaDB and SSL for the node.
[3]
Install Panel + Wings
Both components on the same machine in one go.
[4]
Install phpMyAdmin
Database management GUI served via NGINX with optional SSL.
[5]
Uninstall Panel / Wings
Cleanly remove Panel, Wings, or both — with database cleanup.

How it works

Simple from start to finish

1
Run the command
Paste the curl command into your VPS as root. The script downloads and starts.
2
Answer the questions
Domain, admin email, username, password, SSL preference. All asked upfront.
3
Review the summary
See a full summary of what will be installed. Confirm before anything touches your server.
4
Walk away
PHP, MariaDB, Redis, NGINX, Docker, Wings — all installed and configured automatically.
5
Log in to your panel
Open your domain, log in with the credentials you set. Done.
